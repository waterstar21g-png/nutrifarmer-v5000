## 최종 반영 요약 (V5.0.0.5)

| 구분 | rem / 크기 |
|------|------------|
| 본문 (좌·우·입력·AI답변) | **0.8rem** |
| 제목 (3차·글쓰기 완성 등) | **0.7rem** |
| 버튼·라벨 | **0.6rem** |
| 그리드 설명문 | **0.4rem** |
| AI echo | **0.4rem** |
| AI버튼 | 간격 **0.3rem**, 열·버튼 **80px**, 패딩 **0.5em**, 2차 AI명령 포함 |
| 1차 소재 버튼 | **100px**, 패딩 **0.7em** |
| 모든 버튼 | 타원형, 바탕 `#e8ecf0`, 글자 `#0f2744` **font-weight 800** |
